📊 Superstore Sales Analysis
🔍 Project Overview

This project focuses on analyzing sales data to identify trends, patterns, and key business insights that can support decision-making.

🎯 Objective

To evaluate sales performance across regions, product categories, and time, and identify opportunities for improvement.

🛠️ Tools Used
Power BI
Data Visualization
Data Analysis
📈 Key Insights
Sales show an overall increasing trend with some fluctuations
The West region contributes the highest share of sales
A small number of products drive a large portion of revenue
Some sub-categories show declining sales trends

Learned how to present insights in a clear and visual way
